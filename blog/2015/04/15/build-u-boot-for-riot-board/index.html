Build U-boot for Riot Board

| Comments

1. Get u-boot source

git clone https://github.com/Freescale/u-boot-fslc.git

2. make & build

$cd u-boot-fslc
$export ARCH=arm 
$export CROSS_COMPILE=arm-linux-gnueabihf-
$make riotboard_defconfig 
$make
